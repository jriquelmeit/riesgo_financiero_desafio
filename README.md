# Desafío Riesgo Financiero

Proyecto basado en **node.js** con **Typescript** 

## Requistos 

* Npm 
* Yarn
* Node.js >= 20

## Backend
### Instalación

```bash

cd riesgo_financiero_desafio/backend

#instalar dependencias
yarn install

# copiar variables de entorno
cp env.template .env

```

### Ejecución

```bash

#Levantar servidor local 
yarn run dev

```
